🤖 About The Bot

Truecaller Info Bot is a powerful Telegram bot designed to fetch and display phone number information instantly with blazing-fast performance and advanced backend optimization.

Built for speed, scalability, and monetization, this bot includes premium features, admin controls, and intelligent spam detection.

---

✨ Features

🔍 Core Features

- 📞 Phone Number Lookup (Instant Response)
- ⚡ High-Speed Async Processing
- 🌍 Country Detection with Flags
- 📡 Carrier & Timezone Info

---

👑 Premium System

- ✅ Admin-only premium access control
- ⏳ Custom duration (days-based)
- ♾ Permanent premium option
- ⚡ Faster response for premium users

---

📊 Admin Panel (In-Bot)

- 📤 Broadcast to all users
- 📊 Live stats (users, usage)
- 👑 Manage premium users
- 🚫 Ban / Unban system

---

🧠 Smart System

- 🤖 AI Spam Detection
- 🧠 Intelligent Cooldown System
- ⚡ LRU Cache (Ultra Fast Repeat Queries)
- 🔁 Auto Retry System

---

🎛 User Interface

- 🔘 Button-based UI Panel
- 📱 Clean & Professional Design
- 🖼 Welcome Image + Custom Message

---

🔐 Security & Stability

- 🔐 Secure MongoDB Storage
- 🛡 Anti-Spam Protection
- 🔄 Auto Restart on Crash
- 🧯 Graceful Shutdown System

---

🏗 Tech Stack

- 🐍 Python 3.10+
